Services Module¶

In this project, “services” are mostly implemented as utilities and view-layer logic rather than a dedicated services Django app.

Quote & Invoice Services¶

quotes.utils.generate_quote_pdf(quote)[source]¶

Generate a branded PDF document for a quote.

The PDF includes:

  • Branding header with logo and brand name (if available).

  • Quote metadata (id, created date, status, deadline/timeline).

  • Client details.

  • A single line item based on the estimated amount.

  • An optional project description section.

Parameters:

quote (Quote) – Quote instance to render.

Returns:

Binary PDF content suitable for returning in an HTTP response or writing to disk.

Return type:

bytes

invoices.utils.generate_invoice_pdf(invoice)[source]¶

Generate a branded PDF representation of an invoice.

The PDF includes:

  • Invoice metadata (number, issue/due dates, status).

  • Provider and client details.

  • Line items and monetary totals.

  • Optional notes and payment information.

Parameters:

invoice (Invoice) – Invoice instance to render.

Returns:

Binary PDF content suitable for returning in an HTTP response or writing to disk.

Return type:

bytes
